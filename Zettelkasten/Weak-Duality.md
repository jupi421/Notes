# Weak-Duality

```ad-tip
title: Theorem: Weak Duality
If $f$ is a flow in the network $G$ and there is a s-t cut, then
$$
    |f| \leq c(S,T)
$$
```

```ad-note
title: 
If $|f| = c(S,T)$, then $f$ is a maximum flow and $(S,T)$ is a minimum cut.
```
